README

・If you want to flash the zip file with adb sideload, rename it to rom.zip and place it in this folder.
・If you want to flash a boot image to your boot partition, rename it boot.img and place it in this folder.
・If you want to boot a custom recovery on your device, rename it recovery.img and place it in this folder.
